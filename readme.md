Groot
=====

Gʀᴏᴏᴛ uses genomic data to produce an [N-Rooted Fusion Graph](https://doi.org/10.1093/molbev/mst228).

```
\         /       /
 A       B       C       <-Roots
  \     /       /
   \   /       /
    \ /       /
     AB      /           <-Fusion product
      \     /
       \   /
        \ /
        ABC              <-Fusion product
          \
```
