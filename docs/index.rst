=================================================================
                              Groot                              
=================================================================

`Groot`:t: uses genomic data to produce an `N-Rooted Fusion Graph`_::
    
    \         /       /
     A       B       C       <-Roots
      \     /       /
       \   /       /
        \ /       /
        AB       /           <-Fusion product
          \     /
           \   /
            \ /
            ABC              <-Fusion product
              \
